# Objective

Define Verdict's primary brand color and how it should be used consistently throughout the product.

## Brand Identity

Verdict should no longer use an Apple-inspired blue as its primary accent color.

The official brand accent will be **Electric Lime**.

Reference color:

* **Electric Lime**
* Hex: **#C6FF00**

## Design Philosophy

The product should feel:

* Minimal
* Premium
* Confident
* Modern
* Calm
* Opinionated

The inspiration should evolve from "Apple-inspired" toward:

**Swiss minimalism with subtle fluorescent accents.**

The interface should remain predominantly monochrome (white, black, and grayscale), with Electric Lime used sparingly to create emphasis and reinforce the brand.

The accent color should feel intentional rather than decorative.

## Usage Guidelines

Electric Lime should be used for:

* Primary CTA buttons
* Interactive focus states
* Active navigation states
* Progress indicators
* Loading animations
* Small UI highlights
* Interactive elements requiring emphasis

Avoid using Electric Lime as a general decorative color.

## Verdict Status Colors

The Verdict status system should remain semantic and independent from the brand color.

Continue using:

* 🟢 Launch
* 🟡 Test
* 🔴 Don't Launch

These colors communicate meaning and should not be replaced by the brand accent.

## UI Principles

Maintain:

* Generous whitespace
* Strong typography
* Minimal visual noise
* High accessibility and contrast
* Calm animations
* Premium feel

The overall interface should feel like a clean monochrome canvas with carefully placed fluorescent accents.
