# 002 - Landing Page

## Objective

Replace the default Next.js homepage with Verdict's premium landing page.

## Requirements

- Route: `/`
- Apple-inspired minimal UI
- Large whitespace
- Strong typography
- Headline: "Before you spend, get a Verdict."
- Supporting copy: "AI-powered reviews for static Meta ad creatives before you launch."
- Single primary CTA: "Analyze a Creative"
- CTA navigates to `/analyze`
- No AI integration
- No database
- No auth
- Responsive desktop and mobile design

## Acceptance Criteria

- Default Next.js starter page is fully replaced
- Page looks polished on desktop and mobile
- CTA routes to `/analyze`
- No placeholder content
- App runs with `npm run dev`
- No console errors
